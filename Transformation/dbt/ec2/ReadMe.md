### Activate the environment shared with Airflow
```
source airflow_venv/bin/activate
```


### Install Snowflake addapter
```
python -m pip install dbt-snowflake
```

### Create and enter directory
```
mkdir dbt_proj
cd dbt_proj
```

### Initialize project
```
dbt init
```
